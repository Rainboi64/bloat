تنبيه حول حقوق الملكية الفكرية

عزيزي الزميل/عزيزتي الزميلة،

تتضمن هذه الملاحظات مجهودًا شخصيًا واستثمارًا كبيرًا من الوقت والجهد. لذلك، يُرجى مراعاة ما يلي:

  • هذه الملاحظات مخصصة للاستخدام الشخصي فقط.
  • يمنع منعًا باتًا نسخ أو نقل أو تقديم هذه الملاحظات كعمل شخصي أو مشروع فردي.
  • يرجى احترام حقوق الملكية الفكرية وعدم ارتكاب أي أعمال تعدي على حقوق التأليف والنشر.

الانتحال يعتبر انتهاكًا أكاديميًا جادًا، وقد يؤدي إلى عواقب وخيمة. بدلاً من ذلك، يمكننا التعاون معًا ومشاركة الأفكار بطريقة قانونية وأخلاقية.

شكرًا لتفهمكم واحترامكم, يمان الحلبي.

تعريف المصفوفات

var bookName = ["الخوارزميات وبنى المعطيات", "كتاب آخر", "كتاب ثالث"]
var bookNumber = [520, 321, 123]
var currentQuantity = [3, 5, 2]
var baseQuantity = [10, 7, 3]

المصفوفة هي بنية بيانات بنستخدمها لتخزين مجموعة من العناصر بنفس النوع تحت اسم واحد. العناصر بتكون مرتبة بترتيب معين وكل عنصر إله رقم (index) ببلش من الصفر. المصفوفات بتسهل علينا التعامل مع مجموعة من البيانات بنفس الوقت.

خلينا نشرح المصفوفات الموجودة بالكود تبعنا:

var bookName = ["الخوارزميات وبنى المعطيات", "كتاب آخر", "كتاب ثالث"]

هاي المصفوفة اسمها bookName. بتخزن أسماء الكتب يلي عنا. كل كتاب إله index ببلش من الصفر. مثلاً، "الخوارزميات وبنى المعطيات" هو العنصر الأول وإله index صفر، "كتاب آخر" هو العنصر الثاني وإله index واحد، وهكذا.

var bookNumber = [520, 321, 123]

هاي المصفوفة اسمها bookNumber. بتخزن أرقام الكتب. كل رقم كتاب مطابق لنفس ترتيب اسم الكتاب بالمصفوفة bookName. مثلاً، "الخوارزميات وبنى المعطيات" رقمه 520 وهكذا.

var currentQuantity = [3, 5, 2]

هاي المصفوفة اسمها currentQuantity. بتخزن الكمية الحالية من كل كتاب. مثلاً، عنا 3 نسخ من "الخوارزميات وبنى المعطيات"، 5 نسخ من "كتاب آخر"، و2 من "كتاب ثالث".

var baseQuantity = [10, 7, 3]

هاي المصفوفة اسمها baseQuantity. بتخزن الكمية الأساسية لكل كتاب يلي لازم تكون موجودة بالمكتبة. مثلاً، لازم يكون عنا 10 نسخ من "الخوارزميات وبنى المعطيات"، 7 نسخ من "كتاب آخر"، و3 نسخ من "كتاب ثالث".

هيك منكون شرحنا تعريف المصفوفات وكيفية استخدامهم لتخزين بيانات مرتبة ومرتبة بشكل يسهل علينا التعامل معها.

البحث عن كتاب ما حسب الرقم الخاص بهذا الكتاب

function searchBookByNumber(bookNumberInput):
    for (i = 0; i < length(bookNumber) - 1; i + 1)
        if (bookNumber[i] = bookNumberInput) then
            return bookName[i]
    return "الكتاب غير موجود"

التابع searchBookByNumber هي دالة بتبحث عن اسم كتاب بناءً على رقم الكتاب المدخل (bookNumberInput). الكود بستخدم حلقة for ليتحقق من كل عنصر في المصفوفة bookNumber ويقارن الرقم المدخل بالأرقام الموجودة في المصفوفة. إذا لقى الرقم، بيرجع اسم الكتاب المطابق من المصفوفة bookName. إذا ما لقى الرقم، بيرجع رسالة "الكتاب غير موجود".

خلينا نشرح الكود خطوة بخطوة:

function searchBookByNumber(bookNumberInput):

هون منعرف الدالة searchBookByNumber يلي بتاخد مدخل واحد هو bookNumberInput.

for (I = 0; I < length(bookNumber) - 1; I + 1)

هاي حلقة for، رح تبدأ من I = 0 (العنصر الأول في المصفوفة)، وتستمر طالما I أصغر من طول المصفوفة bookNumber ناقص واحد (حتى نوصل لآخر عنصر في المصفوفة)، وبتزيد I بقيمة 1 في كل دورة.

if (bookNumber[I] = bookNumberInput) then

داخل حلقة for، منستخدم شرط if لنتحقق إذا كان الرقم في المصفوفة bookNumber عند الموقع I يساوي الرقم المدخل bookNumberInput.

return bookName[I]

إذا تحقق الشرط وكان الرقم موجود، منرجع اسم الكتاب المطابق من المصفوفة bookName عند الموقع I.

return "الكتاب غير موجود"

إذا انتهت الحلقة بدون ما نلاقي الرقم المطلوب، منرجع رسالة "الكتاب غير موجود".

function borrowBook(bookNumberInput):
    for (i = 0; i < length(bookNumber) - 1; i + 1)
        if (bookNumber[i] = bookNumberInput) then
            if (currentQuantity[i] > 0) then
                currentQuantity[i] := currentQuantity - 1
                return "تم إعارة الكتاب بنجاح"
            else
                return "لا يمكن إعارة الكتاب، لا توجد نسخ متاحة"
    return "الكتاب غير موجود"

الوظيفة borrowBook هي دالة بتقوم بإعارة كتاب بناءً على رقم الكتاب المدخل (bookNumberInput). الكود بستخدم حلقة for ليتحقق من كل عنصر في المصفوفة bookNumber ويقارن الرقم المدخل بالأرقام الموجودة في المصفوفة. إذا لقى الرقم، بيتحقق إذا كانت الكمية الحالية (currentQuantity) أكبر من صفر. إذا كانت الكمية أكبر من صفر، بيقلل الكمية الحالية بمقدار 1 وبيرجع رسالة "تم إعارة الكتاب بنجاح". إذا ما كانت الكمية الحالية أكبر من صفر، بيرجع رسالة "لا يمكن إعارة الكتاب، لا توجد نسخ متاحة". إذا ما لقى الرقم، ما برجع شيء (يمكن تحسين الكود ليعيد رسالة توضح عدم وجود الكتاب).

function returnBook(bookNumberInput):
    for (i = 0; i < length(bookNumber) - 1; i + 1)
        if (bookNumber[i] = bookNumberInput) then
            if (currentQuantity[i] < baseQuantity[i]) then

                return "تم إعادة الكتاب بنجاح"
